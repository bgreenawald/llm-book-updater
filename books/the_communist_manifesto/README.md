# The Communist Manifesto

## Setup Instructions

1. Get text from Project Gutenberg (https://www.gutenberg.org/ebooks/61) or another public domain source and save to `input_raw.md`
2. Remove frontmatter, TOC, and license (information as per the license)
3. Copy the text to `input_transformed.md`
3. Format the text appropriately (e.g., convert sections to header 2s)
